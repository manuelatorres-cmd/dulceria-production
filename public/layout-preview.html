House rules — content density

  1. Cap content measure. Lists max-w-2xl (640px). Tables max-w-3xl–4xl (768–896px). Grids max-w-5xl (1024px). Never let a row span 1440px.
  2. Cluster, don't span. Secondary meta (channel, qty, due) sits BESIDE or BELOW the primary, not at the far edge. Eye reads vertically in a column, not horizontally across air.
  3. Tabular numbers, right-aligned, fixed columns. Every €/qty/count uses tabular-nums + right-align so digits stack vertically across rows.
  4. Header rows for grids. One small uppercase header (10px, letter-spacing 0.10em) makes the columns self-explaining — no per-row "qty:" / "€" labels needed.
  5. One typeface scale. Body 13–13.5px, sub-meta 11px, h2 19–22px serif, table head 10px caps. Five sizes, used consistently. No mid-row variation.
